## 捣蒜--一站式快速搭建官方网站平台

## 简介
为个人开发者极速建立官方网站，内含主页、api模板页、sdk模板页等可选展示页面。适合中间件开发者、  
前端组建开发者等建立官网。也可导出html代码自行搭建官方网站


## 技术选型

| | |
| --- | --- |
| 开发语言   | Go语言         |
| web框架   | iris          |
| 数据库框架 | gorm          | 
| 数据库     | mariadb       |
| 缓存集群   | dijan         |
| 搜索引擎   | elasticsearch |
| 消息队列   | NSQ           |
| 环境管理   | Docker        |
| 容器编排管理| Kubernetes    |
